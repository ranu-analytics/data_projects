# 📊 Excel Dashboard Project

## 📌 Overview
This project showcases an interactive Excel dashboard designed to analyze and visualize data effectively. It transforms raw data into meaningful insights to support better decision-making.

## 🎯 Key Features
- Interactive dashboard with slicers and filters  
- Data cleaning and transformation  
- Use of Pivot Tables and Pivot Charts  
- Dynamic KPIs and performance metrics  
- User-friendly and visually appealing design  

## 🛠 Tools Used
- Microsoft Excel  
- Pivot Tables  
- Charts & Graphs  
- Conditional Formatting  

## 📈 Insights Generated
- Identified key trends and patterns in data  
- Compared performance across categories/regions  
- Highlighted top-performing segments  



## 🚀 Purpose
The goal of this project is to demonstrate data analysis, visualization, and dashboard-building skills using Excel.
